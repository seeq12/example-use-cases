Compressor Analysis

Workbook Overview

This workbook is divided into multiple worksheets to organize various steps of the analysis. Each worksheet contains a journal with a description of the analysis steps taken on that worksheet. Here is a high level overview of the workbook:

  1. Load and cleanse data
  2. Perform first principal calculations used to analyze compressors
  3. Visualize operation relative to the operating curve on an x-y plot
  4. Flag when operation deviates consistently beyond a given bound
  5. Predict when the compressor needs to be taken offline for maintenance

Starting Signals

  • Discharge Pressure (C23)
  • Discharge Temperature (C23)
  • Suction Pressure (C23)
  • Suction Temperature (C23)
  • Flow (C23)
  • Speed (C23)
