Golden Profiles

Challenge

Golden profiles are a critical tool in assessing process performance and detecting early signs of trouble. Combining disparate datasources and building the profiles at scale can prove to be a challenge.

Solution

Leverage Seeq to combine datasources and build and deploy golden profiles ad-hoc.  Visualize and monitor how well a process is remaining within historical norms. 

Creating Context

Using the available signals, build in the context to define the periods of interest. 

  • Raw Data
  • Pepped Data
  • Context   --> Compare
  • Process

Selecting Reference Data

Select the best (golden) periods using any KPI(s) or by manual selection. 

  • Training

Create Statistical Limits

Build statistical limits using the training dataset. Build an UCL, LCL, and any other set of control limits as needed.

  • Overlay Control Limits --> Swap Assets

Deviation Detection and Monitoring

Identify periods where the temperature deviate from the limits for more than three minutes. Send email notifications during a deviation event. 

  • Deviations -- > Schedule Notifications

Leverage Seeq Data Lab to scale the reference profiles. This example creates a Seeq add-on to enable a wide audience to scale the profiles.
