Creating Boundaries during Grade Transitions

We want to expand our upper and lower boundaries on the viscosity signal horizontally to encapsulate time periods when we are transitioning between product grades

1. To [An invalid workstep link was removed]create a condition for "Grade Transition" use the following formula code to:

  1. Shrink the capsules for each grade by a tiny amount, to allow for a new capsule to be formed between them
  2. Join the shrunken capsules into one composite condition for all grades
  3. Invert this period of all grades to identify only the time periods between grades
  4. Grow this new "between grades" condition
  5. Shift that grown condition to begin at the transition, rather than center around it. 
//Shrunken Capsules $shrunkGR101 = $GR101.shrink(1min) $shrunkGR102 = $GR102.shrink(1min) $shrunkGR103 = $GR103.shrink(1min) $shrunkGR104 = $GR104.shrink(1min) $shrunkGR105 = $GR105.shrink(1min) $shrunkGR106 = $GR106.shrink(1min) $shrunkGR107 = $GR107.shrink(1min) $shrunkGR108 = $GR108.shrink(1min) 
$shrunkGR101      .union($shrunkGR102,5d)      .union($shrunkGR103,5d)      .union($shrunkGR104,5d)      .union($shrunkGR105,5d)      .union($shrunkGR106,5d)      .union($shrunkGR107,5d)      .union($shrunkGR108,5d)      .inverse(1h)      .grow(1h)      .move(1h,1h)

2. [An invalid workstep link was removed]Calculate the maximum and minimum limits to be applied to create boundaries during the "grade transition" capsules. The following code takes the minimum of the LSL signal during the condition "Transition" and creates a new signal with that value over the duration of the condition. 

$LSL.aggregate(minvalue(),$Transition,durationkey())
