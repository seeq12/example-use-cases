Well Depletion Forecast

Potential Savings from Use Case:

Customer M transferred this analysis to Seeq, saving hundreds of FTE hours each month while still helping to determine at scale when to decommission a well or implement an artificial lift.

Problem Statement:

Well decline curves are used to project well production. Often decades worth of data is required to accurately characterize this curve, leading to decline curve analyses being infrequently performed. By performing this analysis continuously,  engineers can characterize their wells and monitor in case actions can be taken to increase production. 

More info available at Decline Analysis Theory and Decline Curve Overview

Analysis:

Data Preparation

Step 1: Access data

  • Data from this well was included in an asset group to allow for asset swapping in later steps

Step 2: Cleanse the rate to remove outliers and negative values

  • The max and agileFilter functions can be appended in Formula for one data cleansing step

Decline Curve Analysis

Step 3: Compare production rate and cumulative production to determine decline curve behavior

  • Calculate cumulative production by trending Well Cycles and integrating production rate in Formula
  • Compare in an XY Plot
    • Note: If the plot is linear, it represents an exponential/logarithmic decline curve
  • Overlay Linear Prediction
    • Note: There is a slight deviation at lower production rates; we can highlight this period and create a condition based on it using Scatter Plot Selection

Step 4: Compare the logarithm of production rate and cumulative production. 

  • If the relationship is linear during the exponential decline curve deviation, the decline curve behaves harmonically at lower production rates
  • If not assume hyperbolic/polynomial decline curve

Well Production Phase and Projection

Step 5: Develop an exponential/logarithmic prediction of production rate based on well life

  • Create a time counter in Formula  based on days during the well cycle
  • Use the Prediction tool to predict the corrected and cleansed production rate based on the time counter. The training window here was included in an asset group to allow for asset swapping in future steps

Step 6: Repeat Step 5 for a hyperbolic/polynomial prediction

Step 7: Combine predictions into one based on the exponential decline curve deviations

Step 8: Characterize well production phase based on variations between decline curves
