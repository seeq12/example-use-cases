Display Results of your Investigation in a Scorecard:

This worksheet calculates the lost financial opportunity that could have been prevented using the model and making reactor process changes in near real time, and displays these metrics in a scorecard.

Display Starting Point

Calculate Scorecard Metrics:

1. Use the Periodic Condition tool to create conditions for annually, quarterly, monthly. (work step)

2. Add the signal for reactor rate to the display pane. (work step)

3. Use Signal from Condition to totalize the volume of product lost during each correctly predicted off-spec event.  (work step)

4. Use formula to multiply the volume by the margin loss associated with product downgrades to get the financial impact of each event. (work step)

//Current Variable Margin difference between on-spec and off-spec product for this unit is $100/mT 

$Volume.divide(2.204623 klb/mT).multiply(100 /mT).setunits("$")
