Bearing Failure Prediction

Objective

Bearings fail exponentially with little notice, making scheduled maintenance difficult. A failure of one bearing can result in the subsequent failure of other system bearings. 

Use vibration data during normal operation to predict bearing failure events.

Starting Point

Raw Data - workstep

Time Range - Feb 11, 2004 - Feb 21, 2004

Analysis Steps

1. Cleanse data (workstep)

  • Identify Bearing Maintenance or downtime periods with Value Search
  • Remove data during down times with Formula

2. Identify normal operation (workstep)

  • Define normal operation as 32 hours after startup with Formula
    • Move the time window by 2 hours to avoid initial startup noise

3. Calculate statistical thresholds (workstep: bearing 1 only or all bearings)

  • Define the max vibration threshold as the average vibration + 3 standard deviations with Formula

4. Flag deviations above thresholds (workstep: bearing 1 only or all bearings)

  • Identify periods of time where the vibration exceeds the threshold for 10 minutes or longer using Value Search
  • Combine flags for all bearings into a single condition using Formula
