How to Create time delayed signals in Seeq:

Use the formula tool and the delay function to delay reactor signals by the residence time between when the reactants enter the reactor and the finished product sample point. In this case, the SME knows that this time is ~1.5 hours. An example of the function syntax is below.

$RxtrExitTemp.delay(1.5h)

How to Forecast a constant value into the future:

We want to project the grade signal into the future as a constant value. This will be used to create the target and boundaries necessary to use the viscosity model in near real time.

$grade.forecastConstant(7d)

Use the "Step to the current time" arrow followed by the "Step forward by half an interval" arrow to see how the forecastConstant() function projects the current value of a signal into the future. (work step)
