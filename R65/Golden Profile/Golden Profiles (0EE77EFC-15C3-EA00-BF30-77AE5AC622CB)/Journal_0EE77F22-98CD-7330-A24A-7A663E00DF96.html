Golden Profiles

Challenge

Golden profiles are a critical tool in assessing process performance and detecting early signs of trouble. Combining disparate datasources and building the profiles at scale can prove to be a challenge.

Solution

Leverage Seeq to combine datasources and build and deploy golden profiles ad-hoc.  Visualize and monitor how well a process remains within historical norms. 

Creating Context

Using the available signals, build in the context to define the periods of interest. 

  • Raw Data
  • Context
  • Process

Selecting Reference Data

Select the best (golden) periods using any KPI(s) or by manual selection. 

  • Training

Create Statistical Limits

Build statistical limits using the training dataset. Build an upper, lower, and any other set of control limits as needed. 

  • Overlay Control Limits --> Swap Assets

Deviation Detection and Monitoring

Identify periods where the temperature deviate from the limits for more than two minutes. Send email notifications during a deviation event. 

  • Profiles and Deviations --> Schedule Notification

Golden Profile Scaling

Leverage Seeq Data Lab to scale the reference profiles. This example accelerator add-on enables a wide audience to scale the profiles across signals and statistics quickly and easily.
