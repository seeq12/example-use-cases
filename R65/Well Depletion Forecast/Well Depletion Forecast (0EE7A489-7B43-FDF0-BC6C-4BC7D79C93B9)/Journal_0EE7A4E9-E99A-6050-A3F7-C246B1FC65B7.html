Well Depletion Forecast

Problem Statement:

Well decline curves are used to project well production. Decades worth of data is required to accurately make this curve, leading to decline curve analyses being infrequently performed. By performing this analysis continuously, engineers can characterize their wells and monitor to suggest actions to increase production. 

More info available at Decline Analysis Theory and Decline Curve Overview

Well Life Projection
 

Solution:

Cleanse the production rate to capture overall trends 
(Do it! Start smoothing the Production Rate using Signal Smoothing, Agile Filter of 3d)

  • Data is available in Asset Group under Various Wells >> Well 1
  • Cleanse signal with Agile Filter

Perform a decline curve analysis to determine the behavior of this well
(Do it! Start highlighting the lower portion of the plot where Cumulative Production > 6000k bbl and create a condition for when to use a different prediction)

  • Calculate Cumulative Production by trending well cycles and integration production rate in Formula
  • Compare in an XY Plot
    • Note: If the plot is linear, it represents an exponential/logarithmic decline curve
  • Overlay a linear regression
    • Note: There is a slight deviation at lower prediction rates. We can highlight this period as the Decline Curve Change and create a condition based on it using Scatter Plot Selection
  • Compare the logarithm of production rate and cumulative production
    • If the relationship is linear during the exponential decline curve deviation, the decline curve behaves harmonically at lower production rates
    • If not, assume hyperbolic/polynomial decline curve

Develop a predictive model to characterize well production

  • Create a time counter in Formula based on days during the well cycle
  • Use the Prediction tool to predict the cleansed production rate based on the time counter. The training window was included in an asset group to allow for asset swapping in future steps
  • Combine exponential and hyperbolic predictions into one based on the exponential decline curve deviations

Characterize well production phases based on these predictions
