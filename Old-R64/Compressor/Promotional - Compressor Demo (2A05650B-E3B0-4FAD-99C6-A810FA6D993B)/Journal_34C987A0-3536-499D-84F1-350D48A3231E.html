X-Y Plots with Compressor Curves

Objective

When operating conditions are held constant (e.g., operating speed, gas composition, flow), we can observe how the efficiency changes over time and moves away from the compressor curve.

Analysis

1. Use Scatter Plot to visualize x-y plots. (workstep)

2. Adjust the color ranges on the mini-map to observe how performance is changing over time. (workstep)

3.  Filter the visual to only show when Speed Between 10k - 10.1k rpm). (workstep)

4. Create the Compressor Curve(s) using Formula.

  • Use a program to translate the compressor curve image into a table of X-Y coordinates. An example is https://apps.automeris.io/wpd/ .
  • Use a regression tool to create a polynomial fit to the table of X-Y coordinates.
  • Enter the polynomial equation into Formula.

5. Add the Compressor Curve(s) to display. (workstep)
