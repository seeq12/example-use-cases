Vapor Cut Estimations

Problem Statement:

In refineries, multiple units work independently to modify operations to achieve desired products based on current demand, often leading to ineffective communication of issues. With Seeq various sources of data can be brought in and analyzed, giving different units one forum through which they can observe relative upstream and downstream processes and make informed decisions. 
In this use case, analyzing lab data under different process conditions based on the ASTM D86 standard allowed engineers to have a better understanding of the distillation being performed in the Crude Distillation Unit (CDU). The results were then communicated to downstream units, giving them the ability to infer how much throughput they will receive and its qualities.

Solution:

Vapor Cut Profile and KPIs
 

Connect to data from various datasources

  • Distillation Colum Process Data: From historians such as OSIsoft, Honeywell PHD, etc.
  • Tank Inventory/Work Order Data: From SQL based systems like Maximo, SAP, etc.
  • Lab Data: From CSV files or LIMS/SQL/historians if available
    • Subset of lab data for demo
    • Full data set

Assign blends based on crude parameters

  • Convert tank feed (delta of daily inventories) capsule properties into signals using .toSignal()
  • Calculate total crude feed based on tank feeds and pipeline flow rate
  • Capture times when we are using our "Standard Blend" in Formula (can also be done with multiple Value Searches and Composite Condition)

Profile lab temperature data to compare recent results to historical

  • Delay lab tags by their corresponding boil percentage in seconds and combine into one temperature profile
    • Results can also be viewed in XY Plot where coloring can be used to show variation between blends or over time
  • Highlight each lab profile using .isValid()
  • Compare profiles in Capsule View
  • Develop a Reference Profile and Value Search to highlight anomalous measurements during Standard Blends
    • Delay lab profiles by -2 days to align with process data and filter to only show profiles during standard blends
    • Use Manual Condition to highlight "good" profile distributions
    • Generate Reference Profiles based on the good profiles and apply to all profiles of Standard Blend
    • Highlight deviations from these good profiles and show limits in scorecard
  • Display in table for easier viewing with % deviation KPI

Determine the percentage and amount of crude that will become vapor and sent to downstream units

  • Use Value Search to determine when the lab profile is less than the side cut temperature of the distillation column
  • Calculate how long the Value Search lasts, representing the percentage that will be vaporized at this temperature (Note: we don't have a (0,0) point so we need to add 0.5 for accuracy)
  • Determine the amount of vapor based on the vapor cut percentage and total crude feed
  • Display in Table for easier viewing

Consolidate results into an automatically updating Organizer Topic Report
