Apply the Model:

Having selected our model and applied the Boundaries tool, Seeq's auto-update features can be used to monitor what is currently going on in the reactor and inform process control moves based on the predicted viscosity.

  1. Add the raw reactor signals back to the display pane. (work step)
  2. Use the "Go to Now" button to change the display date range. 
  3. Use the Auto-updates button to configure the display pane to auto-update ~each minute. 
  4. Use the Predicted Viscosity Model to make decisions about reactor control moves to keep the product viscosity at target. 

Display final view (go to now followed by step forward 1/2 interval)
