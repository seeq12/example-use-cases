Bearing Failure Prediction

Objective

Bearings fail exponentially with little notice, making scheduled maintenance difficult. A failure of one bearing can result in the subsequent failure of other system bearings. 

Identify Failure Events Earlier

Analysis Steps

Bring in bearing data and identify downtime and normal operation.

Calculate statistical thresholds for one bearing, scale to all other bearings.

Flag deviations above thresholds.

Enhance detection with Data Science Input

Share analysis with data scientist with all relevant context - normal operation, downtime, etc. 

The Data Science can connect to the data and pull it into their application of choice using the Seeq Python module and Seeq Data Lab. 

Once they have created a model, they can push the model back into this analysis for the SME to validate.
