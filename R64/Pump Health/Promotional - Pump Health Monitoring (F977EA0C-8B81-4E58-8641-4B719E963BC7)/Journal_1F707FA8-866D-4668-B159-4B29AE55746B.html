Signal Cleansing: 

1. Use Value Search to find when pump was off. (step)

2. Clean Signals.

  • Pump Discharge Flow:Removed samples when pump was off and removed outliers. Applied agile filter with 1 minute period.
$FlowRate.removeoutliers().agileFilter(1 min).setMaxInterpolation(1d).remove($OFF)
  • Pump Discharge Press: Removed samples when pump was off and removed outliers. Applied agile filter with 1 minute period.
$Pressure.remove($OFF).removeoutliers().agilefilter(1min)
  • Recirc Flow:  Spliced value less than 0 with 0. Applied agile filter with 1 minute period.
$negativeFlow = $RecircFlow.valueSearch(isLessThan(0))
$zero = 0.toSignal()
$RecircFlow.splice($zero,$negativeFlow).agileFilter(1min)
  • Suction Temp: Removed samples when pump was off and removed outliers. Applied agile filter with 1 minute period.
$Temp.remove($OFF).removeoutliers().agilefilter(1min)
  • Suction Press:  same as Pump Discharge Pressure
$Pressure.remove($OFF).removeoutliers().agilefilter(1min)
  • Drum Level: Same as Suction Pressure
$Level.removeoutliers().agilefilter(1min)
  • Coverting API to SG using: API = (141.5/SG) -131.5
141.5 /($API + 131.5)
