Solution

Access raw data, bringing in total steam and all tags impacting it. (Do it! start search for 'Line1' in the data tab to find some of the tags used in this analysis)

Cleanse the data, converting string tags to numeric, so they can be used in a regression model. 

Identify uptime, total steam greater than zero. (Do it! start Use Value Search to find when 'Total Steam' is greater than zero, ignore capsules shorter than 1 day)

Create an energy model of total predicted steam. 

Validate the model using a scatterplot over the training window. Trendline: y = 1.11x-1216 (r^2 = 0.900)

Use the regression coefficients to inform continuous improvement efforts. Then, measure the impact. Trend of Energy Improvements OR Table of Monthly Improvements
